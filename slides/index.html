Resolve remote agents via HTTP


AIAgent ResolveA2AAgent(string envVar, string cardPath = "/.well-known/agent-card.json")
{
    var url = Environment.GetEnvironmentVariable(envVar);

    var httpClient = new HttpClient { BaseAddress = new Uri(url) };
    var resolver = new A2ACardResolver(httpClient.BaseAddress!, httpClient, agentCardPath: cardPath);

    return resolver.GetAgentCardAsync().Result.AsAIAgent(httpClient);
}

var weatherAgent = ResolveA2AAgent("services__weather-agent__https__0");

The voice agent is a WebSocket bridge to Voice Live, but it reuses the same specialist agents as tools.

Browserstreams audio over /ws/voice
Voice Livehandles realtime speech turns
ToolsA2A + Foundry agents become voice functions
Tracingtool calls are emitted as voice session spans

app.Map("/ws/voice", async context =>
{
    var agents = context.RequestServices
        .GetRequiredService<Dictionary<string, AIAgent>>();

    var handler = new VoiceWebSocketHandler(
        webSocket,
        credential,
        endpoint,
        model,
        voice,
        systemPrompt,
        agents,
        logger,
        conversationId,
        cosmosContainer);

    await handler.RunAsync(context.RequestAborted);
});
